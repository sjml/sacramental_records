## Sacramental Record Keeping

An utterly basic way for me to log when I've performed a sacrament. A protected webform (just using basic `.htaccess`) and PHP script to log the details to a SQLite database, which then writes out a publicly accessible JSON with the counts. 

This is not fancy; nor is it meant to be. It's deployed as a directory on an Apache server, like we did in days of yore.
